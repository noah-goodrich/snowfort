Snowfort

The only WAF scorecard for Snowflake.

Your Snowflake account is bleeding money in places you cannot see, and exposing data in ways you would notice if you knew where to look. Snowfort scans it and tells you exactly where, scored by pillar, ranked by severity, with a remediation hint on every finding.

One command to install. One command to scan. No agent, no vendor signup, no cost. MIT-licensed open source.

Install

pipx install snowfort-audit

Scan

eval $(snowfort login)
snowfort audit scan

That's it. Five minutes from install to a scored report. Offline mode is available too, for scanning SQL projects in CI before deploy.

How it works

Snowfort runs 164 deterministic rules across all six Snowflake Well-Architected pillars: Security, Cost, Performance, Reliability, Operations, and Governance, plus a Cortex AI Governance pack. Each rule checks one specific configuration or usage pattern. The output is a 0–100 score per pillar, an overall letter grade from A to F, and a JSON manifest that a CI pipeline or downstream agent can act on.

The tool reads ACCOUNT_USAGE and runs SHOW / DESCRIBE queries. It never writes to your account. There is no agent to install, no third-party endpoint to phone home to. It is a Python CLI you run locally.
